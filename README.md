## Features

- Contains over **5,000** emojis from Unicode 15.1 üí™
- Supports:
  - `shorcodes`: **Discord**, **Github**, **Slack** ü´°
  - skin tone variations: üëç üëçüèª üëçüèº üëçüèΩ üëçüèæ üëçüèø
  - hair style variations: ü¶∞ ü¶± ü¶≥ ü¶≤
  - multi-person: üë®‚Äçüë©‚Äçüëß‚Äçüë¶
- Provides [`Emojis`](#emojis-collection) class that has all the info about each emoji:
  - value: üòÜ
  - unicode: 1F606
  - name: Grinning Squinting Face
  - apple name: Grinning Face with Squinting Eyes
  - also known as: ><, Closed-Eyes Smile, Big Grin, Laughing, XD
  - group: Smileys & Emotion
  - subgroup: face-smiling
  - version: 6.0
  - status: fully-qualified
  - shortcodes:
    - CLDR: grinning_squinting_face
    - Discord: laughing, satisfied
    - Github: laughing, satisfied
    - Slack: laughing, satisfied
- Provides [`emojis`](#emojis-extension) extension to interact with emojis within any text. Quick overview of just a few possibilities:

```dart
'textüòÄtextü§¶üèæ‚Äç‚ôÄÔ∏ètext'.emojis

                         .only // false

                         .remove // texttexttext

                         .replace('-') // text-text-text
                         .first
                               .value // üòÄ
                               .unicode // 1F600
                               .name // Grinning Face
                         ... // and many more
                          
// to Slack shortcodes:
'üòÄtextü§¶üèæ‚Äç‚ôÄÔ∏è'.emojis.toSlackShortcodes(); // :grinning:text:woman-facepalming::skin-tone-5:

// from Slack shortcodes:
```
